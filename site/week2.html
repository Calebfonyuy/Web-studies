Introduction to CSS3

In this week, we introduce the Cascading Style Sheets (CSS3). We will look at a brief introduction to this alongside the concept of selectors in HTML and the usage of the stylesheets to format web pages.

Notes

  1. Selectors

    2. Selectors in HTML refer to elements that can be used to reference a particular section of the page in order to apply styles to it. The main selectors used in HTML include:
    • HTML elements: By this we refer to the element associated with the tag we are targetting. Any formatting defined for a given element will be applied to all areas where the associated tag is use.
    • Class: The class of a given section in a HTML is usually set as an attribute of a tag and written as part of the opening tag. For example: <htmltag class="test-class">sample test for htmltag </htmltag> The class name can be reused with several tags so as to apply similar formatting to these sections.
    • ID: This selector is used to uniquely reference a particular section of the page. It is useful when we desire to apply a specific formatting to a certain section of our page. The HTML id attribute is declared exactly like the class attribute with the class keyword replaced by id.

  2. Defining styles

    3. The format for defining a style in stylesheets is as follows: selector{ style-name: value; <!-- Other styles can be found here--> } It should be noted that a particular style defination can contain numerous styles but on the condition that they are distinct.

    When writing your code, the best practice is to place all the styles inside the style tag in usually placed in the head of the html document.

    The format for defining a style using a HTML element as selector is as follows:

    element { style-name: value; <!-- Other styles may be defined here --> }

    To define the style using a class as selector, the format is as follows: .class-name { style-name: value; <!-- Other styles may be defined here --> } In the above illustration, it should be noted that the name of the class does not contain the leading point. That is, the name of the class is class-name but the point is added to indicate that we are selecting a class.

    For style definition using the id attribute, we have the following format:

    #id { style-name: value; <!-- Other styles may be defined here --> }

    In the above notation just like in the class definition, the id is actually id. The # only indicates the fact that our selector is an id attribute.

  3. Combining selectors

    4. Several selectors can be used together especially when it comes to using elements as selectors. However, it is needless to combine id attributes since the latter are specific.

    It is also possible to define a style for several selectors. In such a situation, we write all the selectors before the selectors before the opening curly bracket separated by commas.
