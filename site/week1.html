INTRODUCTION TO HTML5

Brief notes

  1. What is HTML?

    HTML stands for HyperText Markup Language. It is mainly used to define the structure and content of web pages. HTML on its own does not really give us any fancy looking pages (an example is this one lol). It is often combined with Cascading Style Sheets(CSS) and JavaScript to obtain the great and interactive pages we get to use each day.

    HTML is currently being developed by the W3C and WHATWG. The main interest of the W3C is to define standards for HTML as well as HTML versions while the WHATWG takes care of define what can or cannot be done with HTML.

  2. History of HTML

    I will simply give some highlights here about the history of HTML and then direct you to websites that have the complete and rich history.

    • HTML was developed by Tim Berners-Lee between 1989 and 1990. He wrote the browser and server for this in late 1990 and by 1991 published a document on HTML tags defining 18 elements.
    • Between 1991 and 1997, several people took up HTML developement and kept pushing the limits of this field. However, within this period, there were no standards and browsers simply operated according to the desires of their developers. This led to the invention of tags and other situations like different browsers interpreting the same tag differently.
    • In 1997, the World Wide Web Consortium(W3C) came up with the first standards for HTML which defined a new version of it HTML4 which was later developed to get HTML4.01 and subsequently other versions like XHTML
    • Feeling that the development by the W3C was too slow, browser vendors came together to form the Web Hypertext Application Technology Group (WHATWG). This group is in charge of the evolution of HTML but not the versioning.
    • For a complete history of HTML, check out the following links:
      1. Wikepedia
      2. WiReD

  3. HTML elements and tags

    HTML tags

    A tag in html is simply a keyword defining a particular function written between angular bracket. Some examples include

    <p> for paragraph, <b> for bold and <i> for italics

    HTML defines both opening and closing tags. An opening tag simply appears like the tags listed above while for a closing tag we insert a backslash between the first angular bracket and the tag identity. Examples include

    </b>, </i> etc

    It should however be noted that some tags do not have closing tags. Examples are

    <input>, <br>

    HTML elements

    An HTML element is a combination of an opening and a closing tag seperated by the content to which the formatting is being applied. They have the following appearance <p> Paragraph content here </p>
